Project Name: Scientific Calculator

Description:
This project implements a scientific calculator using HTML, CSS, and JavaScript. It provides various mathematical functions such as trigonometric functions, logarithms, square roots, and more.

Usage:
1. Open the HTML file in a web browser.
2. Use the buttons to input mathematical expressions.
3. Press the "=" button to evaluate the expression.
4. The result will be displayed on the screen.

Installation:
No installation is required. Simply open the HTML file in a web browser to use the calculator.

Notes:
- This calculator follows the BODMAS/BIDMAS rule for evaluating expressions.
- Ensure that JavaScript is enabled in your web browser for proper functionality.
